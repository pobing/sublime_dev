# Sublime Text 3 settings and packages

## Use it
```
git clone git@github.com:pobing/sublime_dev.git
cp -r sublime_dev/Packages /Library/Application Support/Sublime Text 3
```
## Packages include
* AdvancedNewFile
* Emmet
* GitGutter
* MarkDown Editing
* ColorPicker
* Ctags
* AutoSave
* SublimeLinter
* AllAutoComplete
* Ruby on Rails snippets
